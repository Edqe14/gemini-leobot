leo is a voice controlled creative agent that can create images, text or video with the power of gemini API. each user can have multiple projects. each project can only have 1 story (import from google docs as markdown), and the app is able to generate character brief nodes based on the story, create the character design inspiration based on the breif and story background, etc. the agent could also told to generate story board based on the story and character design that fits with the user style (node).

all of the app functionality are interactable via voice, similar to jarvis in iron man.
